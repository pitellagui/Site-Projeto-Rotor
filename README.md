Site para o dimensionamelto de um rotor radial baseado em cálculos e correções seguindo modelos físicos. Backend foi feito em python e o visual em html/css/js. Foi usado bibliotecas do pyhton como numpy para o tratamento de dados e barril da ESSS para o tratamento das unidades físicas.

Ultilizei o render para upar o site.
